# CoreCraftGenesis

CoreCraftGenesis ist das öffentliche Genesis‑Portal der CoreCraft‑Engine.  
Es dokumentiert die Genesis‑Struktur, Operator‑Systeme, Ritual‑Frameworks, Payout‑Mechanismen und die gesamte mythologisch‑technische Architektur der Engine.

Die technische Engine selbst befindet sich im Repository **corecraft-chainline**.  
Dieses Portal dient als Dokumentation, Präsentation und Einstiegspunkt für private Chains, Operator‑Registrierungen und Genesis‑Services.

Website: https://corecraftgenesis.ch  
Organisation: https://github.com/corecraftsysteme
